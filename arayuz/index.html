🔌 Geliştirici API Uç Noktaları

Anlık sensör verileri (JSON):

GET http://<ESP32_IP>/api

Örnek yanıt:

{ "humidity": 28.4, "temperature": 31.2, "pump": true, "pumpManual": false }

Pompa kontrolü:

GET http://<ESP32_IP>/pump/on → Pompayı aç (manuel) GET http://<ESP32_IP>/pump/off → Pompayı kapat (manuel) GET http://<ESP32_IP>/pump/auto → Otomatik moda geç
Geliştirici Notu: Bu dosya ESP32 üzerinde çalışan web sunucusunun arayüz taslaklarıdır. Gerçek veriler ESP32'nin IP adresinden JSON olarak çekilebilir (/api endpoint). Eşik değerleri şu an yalnızca ESP32 kodunda değiştirilebilmektedir; ileride /settings endpoint'i eklenebilir.
